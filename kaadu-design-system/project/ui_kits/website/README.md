# Kaadu Website UI Kit

Hi-fidelity recreation of the Kaadu Shopify website for design prototyping.

## Components

| File | Exports | Description |
|---|---|---|
| `Nav.jsx` | `KaaduNav` | Fixed top nav with logo, links, cart |
| `Hero.jsx` | `KaaduHero` | Full-height hero: product image, headline, CTAs |
| `Ticker.jsx` | `KaaduTicker` | Scrolling protest text marquee |
| `ImpactSection.jsx` | `KaaduImpactSection`, `KaaduWomenSection` | Farmer fairness + women's jobs sections |
| `ProductSection.jsx` | `KaaduProductSection` | 3-column product grid |
| `Footer.jsx` | `KaaduFooter` | Footer with links and social |

## Usage

Open `index.html` in a browser — it assembles all components into a clickable prototype.

## Fonts

Both brand fonts must be in `../../fonts/`:
- `Fagies.ttf` — display/H1
- `BanterGrotesk-Regular.woff2` — body/headings

## Design width
1440px desktop. Mobile not yet implemented.
